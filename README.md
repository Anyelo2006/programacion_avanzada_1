# Mis Ejercicios de Programación Avanzada 

Este repositorio contiene una colección de scripts en **Python** desarrollados para la asignatura de Programación Avanzada.

## 📋 Contenido del Repositorio
* **Ejercicio 1:** Registro de datos básicos del estudiante.
* **Ejercicio 2:** Gestión de materias y validación de notas (0.0 a 5.0).
* **Ejercicio 5:** Validación de edad y formato de correo electrónico.

## 🛠️ Tecnologías utilizadas
* Python 3.14
* Git / GitHub para el control de versiones.
