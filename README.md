## Descripcion de Ejercicios

### Ejercicio 1 - Gestion de Notas de Estudiantes
Sistema que permite gestionar las notas de estudiantes usando **diccionarios** y **listas**. Calcula promedios, identifica la nota mas alta y mas baja, y determina si el estudiante aprueba o reprueba.

**Conceptos:** `Diccionarios` `Listas` `Ciclos` `Condicionales`

---

### Ejercicio 2 - Calculadora de Impuestos y Descuentos
Calculadora que aplica impuestos y descuentos a productos. Trabaja con diferentes tasas de impuestos y porcentajes de descuento segun el tipo de producto y monto de compra.

**Conceptos:** `Funciones` `Operaciones matematicas` `Condicionales`

---

### Ejercicio 3 - Generador de Serie Fibonacci
Programa que genera la serie de **Fibonacci** hasta un numero determinado por el usuario. Implementa la logica iterativa para calcular cada termino de la secuencia.

**Conceptos:** `Bucles` `Series numericas` `Logica iterativa`

---

### Ejercicio 4 - Funciones Utilitarias
Coleccion de funciones utilitarias que incluyen:
- **Verificador de numeros primos** - determina si un numero es primo
- **Calculadora de factorial** - calcula el factorial de un numero
- **Contador de vocales** - cuenta las vocales en una cadena de texto

**Conceptos:** `Funciones` `Strings` `Ciclos` `Modularidad`

---

### Ejercicio 5 - Validacion de Datos y Registro de Usuarios
Sistema de registro de usuarios con validacion de datos de entrada. Verifica formato de correo electronico, longitud de contrasena, y otros criterios de validacion antes de registrar al usuario.

**Conceptos:** `Validacion` `Strings` `Manejo de errores` `Entrada de usuario`
